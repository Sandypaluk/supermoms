# 🌸 Supermom

> *A sanctuary built by mothers, for mothers.*

Supermom is a luxury community web app for mothers — designed to connect women in small, intimate groups, and give them easy access to trusted home services. This is a demo built from real research with 13 mothers in Berlin.

---

## ✨ What It Does

**For Moms**
- A warm onboarding survey that gets to know you before you see anything
- 5 curated small-group circles (3–7 members each) you can join
- A services directory with vetted babysitters and home cleaners
- One-tap booking with date, time, and notes

**For Service Providers**
- A separate onboarding flow to register as a babysitter or cleaner
- Profile submission with bio, availability, and experience level

---

## 🏡 The 5 Groups

| Group | Vibe | Format |
|---|---|---|
| 🙏 Praying Moms | Faith & spirituality | Online, Sundays |
| 📖 Bible Study & Discussion | Scripture with a mother's lens | Online, Tuesdays |
| 🌿 The Strolling Sisterhood | Walks, fresh air, real talk | In person, Saturdays |
| 💼 Mama Entrepreneurs | Business-building while mothering | Hybrid, Wednesdays |
| 🎨 Creative Mamas Circle | Journaling, art, reclaiming self | Hybrid, Thursdays |

## 🛠 Tech Stack

| | |
|---|---|
| **Type** | Single-file HTML app |
| **Framework** | Vanilla HTML, CSS, JavaScript — no dependencies |
| **Fonts** | Cormorant Garamond + Jost (Google Fonts) |
| **Photos** | randomuser.me (free public API) |
| **Hosting** | GitHub Pages |
| **API cost** | $0 |

No build step. No npm. No backend. Just open `index.html` in any browser.
